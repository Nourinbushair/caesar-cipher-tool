# Caesar Cipher Encryption Tool

A beginner-friendly cryptography project built with Python to encrypt and decrypt messages using the Caesar Cipher technique.

---

## 🚀 Features

- Encrypt messages
- Decrypt messages
- Custom shift values
- Beginner-friendly implementation

---

## 🛠️ Technologies Used

- Python

---

## ▶️ Run the Project

```bash
python caesar_cipher.py
```

---

## 📸 Example Output

```text
Choose mode: encrypt
Enter message: Hello
Enter shift value: 3

```
